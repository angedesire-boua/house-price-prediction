# 🏠 Prédiction des prix de l’immobilier

## 📌 Description du projet

Cette application web permet d’estimer le prix d’un bien immobilier à partir de ses caractéristiques principales grâce à un modèle de Machine Learning.

L’utilisateur renseigne les informations du logement (surface, nombre de chambres, etc.), et l’application fournit une estimation du prix en temps réel.

## 🎯 Objectif

- Automatiser l’estimation des prix immobiliers  
- Mettre en pratique un projet complet de Data Science  
- Créer une application web interactive avec Streamlit  
- Proposer une interface moderne et intuitive  

---

## 🧠 Modèle de Machine Learning

- Type de modèle : Régression supervisée  
- Algorithme utilisé : Random Forest Regressor  
- Librairies : Scikit-learn, Pandas  

### Variables utilisées :

- Surface  
- Nombre de chambres  
- Nombre de salles de bain  
- Nombre d’étages  
- Parking  
- Climatisation  
- Zone préférée  
- Mobilier  
- etc.

---

## 🖥️ Interface utilisateur

L’application propose :

- Une interface moderne type SaaS  
- Une saisie via sidebar  
- Une estimation dynamique du prix  
- Un résumé des caractéristiques après prédiction  
- Une lecture rapide des facteurs influents  
- Un upload de fichier CSV pour prédictions multiples
